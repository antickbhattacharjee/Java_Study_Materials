# Day 5: Strings

## 📖 Theory
In Java, a String is an object that represents a sequence of characters.

### Core Concepts:
- **String Basics**: Creating strings and understanding their immutability.
- **String Methods**: Common methods like `length()`, `charAt()`, `substring()`, `toLowerCase()`, `toUpperCase()`, and `equals()`.
- **String Manipulation**: Concatenation and comparing strings.
