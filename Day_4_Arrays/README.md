# Day 4: Arrays

## 📖 Theory
An array is a container object that holds a fixed number of values of a single type.

### Core Concepts:
- **1D Arrays**: A linear data structure that stores elements of the same data type.
- **Multidimensional Arrays**: Arrays of arrays, such as 2D arrays (matrices).
- **Array Traversal**: Using loops to iterate through array elements.
