# Day 1: Variables and Datatypes

## 📖 Theory
Java is a statically typed language, which means that every variable must be declared with a data type before it can be used. 

### Core Concepts:
- **Variables**: Containers for storing data values.
- **Datatypes**: Specifies the size and type of values that can be stored. Examples include `int`, `double`, `boolean`, `char`.
- **Operators**: Used to perform operations on variables and values (e.g., Arithmetic operators like `+`, `-`, `*`, `/`).
