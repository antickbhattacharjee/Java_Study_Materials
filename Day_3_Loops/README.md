# Day 3: Loops

## 📖 Theory
Loops are used to execute a block of code repeatedly as long as a specified condition is true.

### Core Concepts:
- **`for` loop**: Best used when you know exactly how many times you want to loop through a block of code.
- **`while` loop**: Loops through a block of code as long as a specified condition is true.
- **`do-while` loop**: A variant of the while loop. This loop will execute the code block once, before checking if the condition is true, then it will repeat the loop as long as the condition is true.
