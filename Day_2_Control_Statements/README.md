# Day 2: Control Statements

## 📖 Theory
Control statements allow you to dictate the flow of execution in your program based on conditions.

### Core Concepts:
- **`if-else` Statements**: Execute a block of code if a condition is true, and another if it's false.
- **`switch` Statements**: Select one of many code blocks to be executed based on the value of a variable.
- **`break` and `continue`**: Used to alter the flow of loops and switch cases.
